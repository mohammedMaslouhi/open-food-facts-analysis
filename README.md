# Open Food Facts Analysis

This project focuses on the analysis of a large-scale dataset from Open Food Facts.

## Objectives
- Clean and prepare raw data
- Handle missing values
- Perform exploratory data analysis (EDA)
- Extract insights from nutritional variables

## Tools
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## Key Steps
- Data cleaning and preprocessing
- Missing values analysis
- Statistical analysis
- Data visualization

## Results
- Improved dataset quality
- Insights on nutritional distributions
